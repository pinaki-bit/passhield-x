# 🛡️ PASSHIELD-X

A world-class, AAA-quality cybersecurity web application inspired by Cyberpunk 2077 and futuristic HUDs. Built with React, FastAPI, TailwindCSS, Framer Motion, and Three.js.

## 🚀 Features
- **3D Particle Background**: Accelerated via Three.js and React Three Fiber.
- **GSAP & Framer Motion**: Smooth, 60 FPS transitions and micro-interactions.
- **FastAPI Backend**: High-performance, asynchronous Python backend for entropy math and vulnerability scanning.
- **Holographic Widgets**: Includes the Entropy Reactor, Threat Radar, Password Genome Map, and Crack-Time Simulator.

## 🛠️ Tech Stack
- **Frontend**: Vite, React, TypeScript, TailwindCSS, Framer Motion, Three.js, Lucide Icons.
- **Backend**: FastAPI, Python, Pydantic, Uvicorn.

## ⚙️ Installation & Deployment

### 1. Start the Backend
Open a terminal and navigate to the backend directory:
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Start the Frontend
Open a new terminal and navigate to the frontend directory:
```bash
cd frontend
npm install
npm run dev
```

### 3. Launch
Open `http://localhost:5173` in your browser.
